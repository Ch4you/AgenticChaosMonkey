## Support

### Where to ask questions
- GitHub Issues: use for bugs and feature requests.
- Discussions/FAQ: use for usage questions and troubleshooting (if enabled).

### What to include in a bug report
- Reproduction steps
- Expected vs actual behavior
- Logs (redacted)
- Environment (OS, Python, dependency versions)
- Chaos plan file and relevant config

### Security issues
Do **not** file a public issue. See `SECURITY.md`.

### Support scope
We support:
- Latest `main`
- Most recent tagged release

Anything older may be closed unless it is a critical security issue.
